How to read this chart

-- Pending = docs received minus docs output. Your backlog.

- - In/Out = received / output ratio. 1.0x = balanced, >1x = backlog growing.

Pending flat at 0 -- healthy, every doc processed

Pending rising + In/Out > 1x -- input outpacing output

In/Out < 1x -- catching up on backlog

In/Out = 1x + Pending flat -- steady state
