Activity Timeline

How to read this chart

■ Pending shows the doc backlog (docs in − docs out).

┅ Fail % shows what percentage of output attempts are errors.

Pending flat at 0 → healthy, every doc processed

Pending rising + Fail % high → errors are causing the backlog

Pending rising + Fail % low → system is slow, not failing

Pending dropping → system is catching up
